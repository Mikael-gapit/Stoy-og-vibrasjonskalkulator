# Støy og vibrasjonsassistent (Bravida)

✅ Fungerer offline og kan installeres som app (PWA)  
📸 Viser bilder av verktøy  
🌍 Støtter 6 språk: NO, SE, DK, FI, EN, PL  
⚠️ Viser advarsler for høy støy (>85 dB) og vibrasjon (>10 m/s²)  
📚 Innebygd HMS-side med Bravidas håndbok og eksterne kilder  
🧾 Eksport av dagsrapport som tekst og PDF  
📱 Tilpasset mobil og hjemskjerm

## Bruk

1. Pakk ut ZIP-filen
2. Last opp innholdet til GitHub Pages, Vercel eller Netlify
3. Åpne `index.html` for testing lokalt

**Tips:** På mobil – legg til på hjemskjerm for app-opplevelse!
